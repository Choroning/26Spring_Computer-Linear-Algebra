## 2-A. $A\mathbf{x}=\mathbf{b}$ 해의 세 가지 경우 (정방 중심)

| 경우 | 조건 | 형태 |
|:---|:---|:---|
| **유일 해 (unique)** | $A$ 가역, rank $= n$ | $\mathbf{x} = A^{-1}\mathbf{b}$ |
| **해 없음 (no solution)** | $\mathbf{b} \notin C(A)$ | — |
| **무한히 많은 해 (infinite)** | rank $<n$, $\mathbf{b}\in C(A)$ | $\mathbf{x}_p + \alpha X$ 등 |

## 2-B. 소거법 (Gaussian elimination) + 후진 대입 (back-substitution)

**소거 단계**: 행 $i$에서 행 $j$의 $l_{ij}$배를 빼서 피벗 아래에 0 만들기
$$\text{row } i \leftarrow \text{row } i - l_{ij} \cdot \text{row } j$$

**흐름**: $(A \mid \mathbf{b}) \xrightarrow{\text{소거}} (U \mid \mathbf{c}) \xrightarrow{\text{후진 대입}} \mathbf{x}$

**후진 대입**: $U\mathbf{x}=\mathbf{c}$를 마지막 행부터 위로 풀어 올라감
- $x_n = c_n / u_{nn}$, $x_{n-1} = (c_{n-1} - u_{(n-1)n}x_n)/u_{(n-1)(n-1)}$, ...

**피벗 (pivot)**: $U$ 대각의 0이 아닌 원소. 0이면 **행 교환 (row exchange)** 필요.

### 왜 행 연산이 정당한가 — 허용되는 3가지 (elementary row operations)

행 = 방정식이고, 이 세 연산은 **해집합을 바꾸지 않음**:

1. 한 행에 **0이 아닌 상수 곱하기** (식 전체에 곱하는 것과 같음)
2. 한 행에 **다른 행의 배수를 더하거나 빼기** (소거법이 쓰는 연산)
3. **두 행 바꾸기** (순서만 바뀌므로 해 불변)

> 🔑 소거법은 (2)를 반복하고, 피벗이 0이면 (3)으로 피해가는 것.

## 2-C. 소거 행렬 $E_{ij}$과 $A = LU$

- **소거 행렬 (elimination matrix)**: 단위행렬에서 $(i,j)$ 위치만 $-l_{ij}$
- **한꺼번에**: $E_{n2}\cdots E_{32} E_{31} E_{21} A = U$, 즉 $EA = U$
- **역변환**: $A = E^{-1} U = LU$

$$\boxed{A = LU}$$

- $L$ = 하삼각 (lower triangular), 대각은 1, **승수 $l_{ij}$가 그 위치에 그대로** 들어감 ★
- $U$ = 상삼각 (upper triangular), 대각은 피벗
- **성립 조건**: 행 교환 불필요 (모든 좌상단 $k\times k$ 부분행렬이 가역)

### ⚠ 승수 $l_{ij}$ 부호 규칙 (시험 실수 포인트)

**정의 기준**: "$R_i \leftarrow R_i - l_{ij} R_j$" — 항상 **빼기** 기준.

| 실제 한 연산 | 승수 값 |
|:---|:---|
| $R_i - 2R_j$ | $l_{ij} = 2$ |
| $R_i + 2R_j$ | $l_{ij} = -2$ (더하면 승수는 음수!) |
| $R_i - (-3)R_j$ = $R_i + 3R_j$ | $l_{ij} = -3$ |

**실전 계산법**: $l_{ij} = \dfrac{\text{없앨 값}}{\text{피벗 값}}$ — 이 결과를 부호 그대로 $L$의 $(i,j)$에 꽂기.

### Singular 행렬의 $LU$ — 되긴 하나?

- **운 좋은 Singular** (종속성이 뒤쪽 열에 몰림): $LU$ 형태로 **분해 가능**. 단 $U$ 대각에 0 생김 → $Ax=b$는 못 풀어
- **운 나쁜 Singular** (앞쪽 열이 종속): 소거 중간에 2열 피벗 자체가 없음 → **표준 $LU$ 불가**
- 결론: "분해가 존재하는가"와 "분해로 문제를 풀 수 있는가"는 별개. Full-rank여야 $LU$가 **실용적으로** 쓸모 있음

## 2-D. $PA = LU$ — 행 교환 필요 시

- **치환 행렬 (permutation matrix) $P$**: $I$의 행을 재배치, $n!$가지
- 성질: $P^{-1} = P^T$, $P$의 열들이 직교
- **부분 피벗팅 (partial pivoting)**: 각 열에서 가장 큰 원소를 피벗으로 → 반올림 오차 최소화, $|L_{ij}|\le 1$

## 2-E. 역행렬 (inverse) 성질

- **정의**: $A^{-1}A = AA^{-1} = I$
- **존재 조건**: $A$가 $n$개의 독립 열을 가질 때 (rank $= n$)
- **유일성**: $BA = I$이고 $AC = I$이면 $B = C$
- **역순 규칙 (reverse order)**: $(AB)^{-1} = B^{-1}A^{-1}$, $(ABC)^{-1} = C^{-1}B^{-1}A^{-1}$
- **$2\times 2$ 공식**:
$$\begin{pmatrix}a&b\\c&d\end{pmatrix}^{-1} = \frac{1}{ad-bc}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$$
여기서 $ad-bc$ = **행렬식 (determinant)** $\det A$
- **삼각 행렬의 역**: 대각 원소가 모두 0이 아니면 역 존재, 역도 삼각 (삼각성·대각 1 유지)

## 2-F. 가우스-조르단 (Gauss–Jordan) — 명시적 $A^{-1}$ 계산 ★

$$\boxed{(A \mid I) \xrightarrow{\text{행연산}} (I \mid A^{-1})}$$

**왼쪽이 단위행렬 $I$가 되도록 행연산하면, 오른쪽에 자동으로 $A^{-1}$이 나타난다.**

### 2-F-1. 왜 성립하는가 — 기본행렬(elementary matrix) 관점

행연산은 전부 **"어떤 행렬을 왼쪽에 곱하는 것"** 과 동치:

- $R_i \leftarrow R_i - l \cdot R_j$ → 어떤 $E$를 왼쪽에 곱하기
- 행 교환 → 치환행렬 $P$를 왼쪽에 곱하기
- $R_i \leftarrow c \cdot R_i$ → 대각행렬 $D$를 왼쪽에 곱하기

$(A \mid I)$에서 시작해 연산을 $k$번 하면:

$$E_k \cdots E_2 E_1 \cdot (A \mid I) = (\,E_k \cdots E_1 A \mid E_k \cdots E_1 \cdot I\,)$$

왼쪽 블록이 $I$가 됐다는 건 $E_k \cdots E_1 A = I$. 이는 곧 $E_k \cdots E_1 = A^{-1}$.
오른쪽 블록은 $E_k \cdots E_1 \cdot I = A^{-1}$로 자동 완성.

> 🔑 양쪽에 **똑같은** 행연산을 적용하기 때문에, 왼쪽이 $I$가 될 때 오른쪽이 $A^{-1}$로 "증명되는" 구조.

### 2-F-2. n×n 일반 행렬 표준 루틴

```
for k = 1 to n:
    # (1) 피봇 확보 (pivot)
    if A[k][k] == 0:
        아래 행 중 A[j][k] ≠ 0인 j를 찾아 R_k ↔ R_j 교환
        (못 찾으면 → 특이행렬, 종료)

    # (2) 피봇을 1로 스케일링 (scaling)
    R_k ← R_k / A[k][k]

    # (3) k열의 다른 모든 행 성분을 0으로 (elimination)
    for i = 1 to n, i ≠ k:
        R_i ← R_i − A[i][k] · R_k
```

핵심 포인트:

- $n \times n$ 행렬은 **피봇을 $n$번 만든다** — 대각선을 따라 내려가며 반복
- 매 스텝의 목표는 **그 열에서 피봇 = 1, 나머지 = 0**
- "**(1,1)만 1로 만들고 시작하면 끝"이 아니라, 대각선 따라 $n$번** 반복
- 어느 순간 피봇을 못 찾으면(아래 행이 모두 0) → **특이행렬, 역행렬 없음**

### 2-F-3. 세 도구의 자유 조합

**목표는 하나**: 왼쪽을 단위행렬로. 수단은 세 가지 — 스케일링, 행 교환, 소거.

- 세 연산 모두 "양변에 똑같이 적용되는 정당한 행연산"
- 어떤 조합으로 써도 결과는 동일한 $A^{-1}$로 수렴
- 순서는 자유: 계산 편한 지름길 발견 시 (예: 분수 피하려고 $R_2 \leftarrow 2R_2 - R_1$처럼 스케일링·소거 결합) 써도 됨
- **최종적으로 왼쪽이 $I$가 되면 장땡**

### 2-F-4. 하삼각행렬 예시 (간단 케이스)

$$S = \begin{pmatrix} 1 & 0 & 0 \\ 1 & 1 & 0 \\ 1 & 1 & 1 \end{pmatrix}$$

대각이 모두 1이라 스케일링 불필요, 소거만 하면 됨:

```
| 1 0 0 | 1 0 0 |       R2 ← R2 − R1      | 1 0 0 |  1 0 0 |
| 1 1 0 | 0 1 0 |  ──►  R3 ← R3 − R1  ──► | 0 1 0 | -1 1 0 |
| 1 1 1 | 0 0 1 |       R3 ← R3 − R2      | 0 0 1 |  0 -1 1 |
```

$$S^{-1} = \begin{pmatrix} 1 & 0 & 0 \\ -1 & 1 & 0 \\ 0 & -1 & 1 \end{pmatrix}$$

> 💡 **삼각행렬 특성**: 하삼각의 역은 하삼각, 상삼각의 역은 상삼각. 대각 1이면 역도 대각 1. 답 꼴이 예상되니 실수 방지에 유용.

### 2-F-5. 일반 3×3 예시 (피봇 확보 + 스케일링 + 소거)

$$A = \begin{pmatrix} 0 & 2 & 1 \\ 1 & 1 & 0 \\ 2 & 0 & 1 \end{pmatrix}$$

**Step 1: 1열 정리** — (1,1) = 0이라 R1 ↔ R2 교환:
```
[ 1 1 0 | 0 1 0 ]
[ 0 2 1 | 1 0 0 ]
[ 2 0 1 | 0 0 1 ]
```
R3 ← R3 − 2·R1로 1열 소거 완료.

**Step 2: 2열 정리** — 피봇 (2,2) = 2를 1로: R2 ← R2/2. 이후 R1, R3에서 2열 성분 소거.

**Step 3: 3열 정리** — 피봇 스케일링 → R1, R2에서 3열 성분 소거.

**결과**:
$$A^{-1} = \begin{pmatrix} -1/4 & 1/2 & 1/4 \\ 1/4 & 1/2 & -1/4 \\ 1/2 & -1 & 1/2 \end{pmatrix}$$

### 2-F-6. 시험장 검산

$A \cdot A^{-1} = I$ 인지 곱해봐서 확인. 1분이면 됨. 실수 잡는 데 가장 효과적.

## 2-G. 가역성 동치 조건 ★ (Invertibility — equivalent conditions)

정방 $A$에 대해 아래는 모두 **동치 (equivalent)** — 하나 성립하면 전부 성립:

- $A$ **가역 (invertible)** = 정칙 (nonsingular)
- $\mathrm{rank}(A) = n$ (full rank)
- 열이 **선형독립 (linearly independent)** / 행이 LI
- $N(A) = \{\mathbf{0}\}$ (trivial null space)
- $A\mathbf{x}=\mathbf{0}$의 유일 해가 $\mathbf{x}=\mathbf{0}$ (unique trivial solution)
- 모든 $\mathbf{b}$에 대해 $A\mathbf{x}=\mathbf{b}$가 **유일 해 (unique solution)**
- $A$가 $n$개의 0이 아닌 피벗 (nonzero pivots)을 가짐
- $\det(A) \ne 0$
- $C(A) = \mathbb{R}^n$
- $A^T$ 가역
- 고유값 중 0이 **없음** (no zero eigenvalue)

## 2-G2. 특이행렬(singular) 심화 — 형태별 판별 ★

**특이행렬 (singular matrix)** = 비가역행렬 = 역행렬 없는 정방행렬.

### 한 줄 핵심

> **"정방 + full rank일 때만 역행렬 존재"** — 이 한 줄이 2-G 전체를 함축.

### 2-G2-1. 어떤 모양이 특이행렬인가

**(1) 영행 또는 영열이 있는 경우**

$$\begin{pmatrix} 1 & 2 & 3 \\ 0 & 0 & 0 \\ 4 & 5 & 6 \end{pmatrix}$$

2행이 영행 → 자명하게 특이.

**(2) 두 행(열)이 완전히 같은 경우**

$$\begin{pmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 1 & 2 & 3 \end{pmatrix}$$

R1 = R3 → 선형종속 → 특이.

**(3) 한 행이 다른 행의 상수배**

$$\begin{pmatrix} 1 & 2 & 3 \\ 2 & 4 & 6 \\ 7 & 8 & 9 \end{pmatrix}$$

R2 = 2·R1 → 선형종속 → 특이.

**(4) 한 행이 다른 행들의 선형결합 — 가장 함정 많음** ⚠

$$\begin{pmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 5 & 7 & 9 \end{pmatrix}$$

R3 = R1 + R2. 눈에 잘 안 띄지만 선형종속 → 특이. **시험 단골**.

### 2-G2-2. 가우스-조르단으로 특이행렬 판정하기

소거 진행 중 **어떤 행이 전부 0이 되면 그 순간 "역행렬 없음" 확정**, 더 진행 불필요.

예 — 위 (4)번 행렬:

```
| 1 2 3 | 1 0 0 |
| 4 5 6 | 0 1 0 |
| 5 7 9 | 0 0 1 |

R2 ← R2 − 4R1, R3 ← R3 − 5R1:
| 1  2  3 |  1 0 0 |
| 0 -3 -6 | -4 1 0 |
| 0 -3 -6 | -5 0 1 |

R3 ← R3 − R2:
| 1  2  3 |  1 0 0 |
| 0 -3 -6 | -4 1 0 |
| 0  0  0 | -1 -1 1 |   ← 왼쪽이 전부 0!
```

→ 특이행렬 확정.

### 2-G2-3. 2×2 특수 공식

$$A = \begin{pmatrix} a & b \\ c & d \end{pmatrix} \quad\Longrightarrow\quad \det A = ad - bc$$

- $ad - bc = 0$ ⟺ 특이
- $ad - bc \ne 0$ ⟺ 정칙, $A^{-1} = \frac{1}{ad-bc}\begin{pmatrix} d & -b \\ -c & a \end{pmatrix}$

### 2-G2-4. 기하학적 의미

$n \times n$ 행렬을 $\mathbb{R}^n$의 선형변환으로 보면:

- **정칙행렬**: 공간을 일대일로 보냄 → 역변환 가능 → $A^{-1}$ 존재
- **특이행렬**: 공간을 **더 낮은 차원으로 납작하게 눌러버림** → 정보 손실 → 되돌릴 수 없음

3×3 특이행렬은 3차원 공간을 평면이나 직선으로 눌러. 납작해진 점들을 원래 3D 위치로 복원하는 건 불가능.
→ **$\det = 0$** 이 **"부피가 0으로 찌그러짐"** 을 뜻하는 이유.

### 2-G2-5. 시험장 빠른 판별 체크리스트

1. **영행/영열 눈으로 스캔** → 있으면 즉시 "역행렬 없음"
2. **두 행이 같거나 배수 관계** → 있으면 즉시 "역행렬 없음"
3. **2×2면 $ad-bc$** 바로 계산
4. **3×3 이상이면 가우스-조르단** 진행 중 영행 나오면 그 순간 stop

## 2-H. 계산 비용 (computational cost)

- $A \to U$ 소거: $\tfrac{1}{3}n^3$
- $\mathbf{b} \to \mathbf{c} \to \mathbf{x}$ (전진 + 후진 대입): $n^2$
- 행렬 곱: $n^3$
- **명시적으로 $A^{-1}$을 구한 뒤 $A^{-1}\mathbf{b}$ 곱하는 건 낭비** — 소거법으로 바로 풀 것

---

## 2-I. 전치 (transpose) — $A^T$

- **정의**: $(A^T)_{ij} = A_{ji}$. $A$의 $i$행이 $A^T$의 $i$열이 됨
- **전치의 진짜 의미**: $(A\mathbf{x})\cdot\mathbf{y} = \mathbf{x}\cdot(A^T\mathbf{y})$
- **기본 규칙**:
  - $(A+B)^T = A^T + B^T$
  - $(AB)^T = B^T A^T$ — **역순 (reverse order)**
  - $(A^{-1})^T = (A^T)^{-1}$

## 2-I2. 전치 심화 ★

### 2-I2-1. 크기 변화

$m \times n$ 행렬을 전치하면 $n \times m$ 행렬.

- $3 \times 4$ → $4 \times 3$
- $n \times n$ 정방 → $n \times n$ (크기 유지)

### 2-I2-2. 구하는 직관 두 가지

**방법 1: 1행 = 1열로 내리기**

$$A = \begin{pmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{pmatrix} \xrightarrow{\text{전치}} A^T = \begin{pmatrix} 1 & 4 \\ 2 & 5 \\ 3 & 6 \end{pmatrix}$$

**방법 2: 주대각선 기준 "접어서" 대칭 (정방일 때 직관적)**

$$\begin{pmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 7 & 8 & 9 \end{pmatrix} \xrightarrow{\text{주대각선 대칭}} \begin{pmatrix} 1 & 4 & 7 \\ 2 & 5 & 8 \\ 3 & 6 & 9 \end{pmatrix}$$

대각성분은 그대로, 나머지만 대각선 너머로 튕김.

### 2-I2-3. 조건 없음 ★

**전치는 어떤 행렬이든 항상 가능.**

- 정방 / 비정방 OK
- full rank / rank-deficient OK
- 영행렬도 전치 가능 (여전히 영행렬)

역행렬이 "정방 + full rank"라는 까다로운 조건을 요구하는 것과 대조적. 전치는 그냥 "행·열 맞바꿈"이라 조건 없음.

### 2-I2-4. 성질 총정리

| 성질 | 식 |
|:---|:---|
| 이중 전치 | $(A^T)^T = A$ |
| 합의 전치 | $(A + B)^T = A^T + B^T$ |
| 스칼라 배 | $(cA)^T = c A^T$ |
| **곱의 전치 (중요!)** | $(AB)^T = B^T A^T$ |
| 여러 개 곱 | $(ABC)^T = C^T B^T A^T$ |
| 역의 전치 | $(A^{-1})^T = (A^T)^{-1}$ |
| 행렬식 | $\det(A^T) = \det(A)$ |
| 랭크 | $\text{rank}(A^T) = \text{rank}(A)$ |

**곱의 전치 순서 뒤집힘** 직관: $A$가 $m\times n$, $B$가 $n\times p$일 때 $(AB)^T$는 $p\times m$. 크기 맞추려면 $B^T A^T$ 순서일 수밖에 없음.

### 2-I2-5. 왜 전치를 배우는가 (ML/AI 연결)

- **내적**: $\mathbf{x} \cdot \mathbf{y} = \mathbf{x}^T \mathbf{y}$
- **이차형식**: $\mathbf{x}^T A \mathbf{x}$ (최적화)
- **최소제곱법**: $\hat{\beta} = (X^T X)^{-1} X^T y$ — 선형회귀 정규방정식
- **공분산 행렬**: $\Sigma = \frac{1}{n} X^T X$
- **그람 행렬 (Gram matrix)**: $A^T A$ — 항상 대칭 & 양의 준정부호

## 2-J. 대칭 행렬 (symmetric matrix)

- **정의**: $S^T = S$
- $S^{-1}$도 대칭
- $A^T A$, $AA^T$는 **항상 대칭**
- $A^T A$가 가역 $\iff$ $A$의 열들이 선형독립
- **$S = LDL^T$ 분해**: $S$ 대칭일 때 $LU$보다 더 자연스러움 (대칭성 보존, 소거 비용 절반)
- 고유값이 **실수** 보장 (중요한 이론적 성질)

## 2-J2. 전치로 정의되는 특수 행렬 — 반대칭·직교 ★

### 반대칭 행렬 (skew-symmetric)

**정의**: $A^T = -A$

$$\begin{pmatrix} 0 & 2 & -3 \\ -2 & 0 & 5 \\ 3 & -5 & 0 \end{pmatrix}$$

- **주대각선은 반드시 0** ($a_{ii} = -a_{ii}$ → $a_{ii} = 0$)
- (i,j)와 (j,i) 성분이 부호 반대
- 외적 $(\mathbf{u}\mathbf{v}^T - \mathbf{v}\mathbf{u}^T)$가 이 꼴

### 직교 행렬 (orthogonal matrix) ★

**정의**: $Q^T Q = Q Q^T = I$, 즉 $\boxed{Q^T = Q^{-1}}$

- **전치가 곧 역행렬!** 계산상 엄청난 이점
- 열들이 서로 **직교 + 단위벡터** (정규직교 기저, orthonormal basis)
- 예: 회전 행렬 (rotation matrix), 반사 행렬 (reflection)
- 성질: $\|Q\mathbf{x}\| = \|\mathbf{x}\|$ (길이 보존), 각도 보존
- $\det Q = \pm 1$

> 🔑 일반적으로 $A^T \ne A^{-1}$. **직교행렬일 때만** $A^T = A^{-1}$.

## 2-K. 유한 차분 행렬 $K, T, B$ (Ch 2.5)

| 이름 | 형태 | 성질 |
|:---|:---|:---|
| $K$ (고정-고정, fixed-fixed) | $(-1, 2, -1)$ **삼중대각 (tridiagonal)** | 대칭, 가역, **양의 정부호 (positive definite)**; $\tfrac{1}{h^2}K\mathbf{u}=\mathbf{f}$ |
| $T$ (자유-고정, free-fixed) | 왼쪽 위 $1$로 시작 | 가역, $T = LL^T$ (촐레스키, Cholesky) |
| $B$ (자유-자유, free-free) | 양쪽 모두 자유 | **특이 (singular)**, $B\mathbf{1}=\mathbf{0}$ |

- **양의 정부호 (positive definite)**: 대칭 + $n$개의 **양의 피벗** + 모든 $\mathbf{x}\ne\mathbf{0}$에 대해 $\mathbf{x}^T K\mathbf{x}>0$
- **양의 준정부호 (positive semi-definite)**: $\mathbf{x}^T K\mathbf{x}\ge 0$, 피벗 $\ge 0$

**유한 차분 공식**:
- 전진 차분 (forward): $\frac{y(x+h)-y(x)}{h}$, $O(h)$
- 후진 차분 (backward): $\frac{y(x)-y(x-h)}{h}$, $O(h)$
- **중심 차분 (centered)**: $\frac{y(x+h)-y(x-h)}{2h}$, $O(h^2)$ ★
- 이차 차분 (second): $\frac{y(x+h)-2y(x)+y(x-h)}{h^2}$, $O(h^2)$

## 2-L. 테일러 급수 (Taylor series) — 문제 풀 때 필수

### 일반 공식 (시험지 여백에 먼저 적어둘 것)

$$y(x_0+h) = y + hy' + \frac{h^2}{2}y'' + \frac{h^3}{6}y''' + \frac{h^4}{24}y^{(4)} + \cdots$$

$$y(x_0-h) = y - hy' + \frac{h^2}{2}y'' - \frac{h^3}{6}y''' + \frac{h^4}{24}y^{(4)} - \cdots$$

> 분모는 팩토리얼 $1!, 2!, 3!, 4!, 5! = 1, 2, 6, 24, 120$
> **홀수 차수만 부호 뒤집힘** → 더하면 홀수 소거(짝수만), 빼면 짝수 소거(홀수만)

### 자주 쓰는 3개 (암기)

$$\sin h = h - \frac{h^3}{6} + \frac{h^5}{120} - \cdots\quad(\text{홀수만, 부호 교대})$$

$$\cos h = 1 - \frac{h^2}{2} + \frac{h^4}{24} - \cdots\quad(\text{짝수만, 부호 교대})$$

$$e^h = 1 + h + \frac{h^2}{2} + \frac{h^3}{6} + \frac{h^4}{24} + \cdots\quad(\text{모든 차수, 전부 +})$$

### ⭐ 정확도 판정 — 실전 3단계

1. 주어진 함수의 **Taylor 전개 쓰기**
2. **(진짜값) − (근사값)** 계산 → 앞부분이 상쇄되고 오차 항만 남음
3. **남은 항의 최저 차수 = 정확도**

### 한 줄 규칙 ★

> **"근사식이 Taylor의 앞 $k$개 항까지 포함 → 버린 첫 항의 차수가 정확도"**

**예시**:
| 근사 | 포함된 항까지 | 버린 첫 항 | 정확도 |
|:---|:---|:---|:---:|
| $\sin h \approx h$ | $h$ | $h^3$ | $O(h^3)$ |
| $\cos h \approx 1 - h^2/2$ | $h^2$ | $h^4$ | $O(h^4)$ |
| $e^h \approx 1+h$ | $h$ | $h^2$ | $O(h^2)$ |
| $e^h \approx 1+h+h^2/2$ | $h^2$ | $h^3$ | $O(h^3)$ |

> ⚠ 자주 하는 실수: "오차 식에 $h$ 있으니 1차" — **틀림**. $h$가 **몇 제곱부터** 시작하느냐가 차수.

### 유한 차분 공식 정확도 (위 규칙 적용)

- 전진 차분: $O(h)$
- 후진 차분: $O(h)$
- **중심 차분**: $O(h^2)$ ★ — 뺄셈으로 짝수 차수 소거
- 이차 차분: $O(h^2)$ — 덧셈으로 홀수 차수 소거

### 다항식 대입 검증법 (Q20 유형)

**"차분 공식이 $k$차 정확도"를 보이는 법**:
- 중심점을 $x = 0$으로 잡기
- $u = x^2, x^4, \ldots$ 대입해 공식값 = 참값 확인
- $u = x^k$까지 정확히 맞추면 **$k$차 정확도 이상** 보장
- 대칭적 공식이면 **홀수 다항식은 자동 통과**, 짝수만 테스트

## 2-L2. 테일러·차분 문제가 선형대수 책에 있는 이유

- **미분방정식 $-u'' = f$** → **차분 공식으로 이산화** → **행렬 $K\mathbf{u}=h^2\mathbf{f}$** 형태로 등장
- $K, T, B$는 **차분 공식을 행렬로 포장한 것**
- 차분 공식의 정확도 = 행렬 해의 정확도
- 앞에서 배운 도구 ($LDL^T$, 대칭성, tridiagonal 소거의 $O(n)$ 속도 등)이 여기서 **실전 활용**

## 2-O. 비정방 행렬의 "역행렬 비슷한 것" ★ (보충)

$m \times n$, $m \ne n$인 행렬은 **일반 역행렬이 존재하지 않음** (구조적으로 불가능). 대신 한쪽만 성립하는 역 개념과 가장 일반화된 pseudoinverse가 있음.

### 2-O-1. 왜 비정방은 역행렬이 없는가

$A$가 $3 \times 4$라면 $A^{-1}$이 있다고 가정하면 $4 \times 3$이어야 함. 하지만:
- $A A^{-1}$은 $3 \times 3$ → $I_3$ 가능
- $A^{-1} A$는 $4 \times 4$ → $I_4$와 같으려면 rank 4 필요
- 그런데 $A^{-1} A$의 rank ≤ rank($A$) ≤ 3 (왜냐하면 $3\times 4$의 rank는 최대 3)
- → $A^{-1}A = I_4$는 **불가능**

결론: **$AA^{-1} = I$ 와 $A^{-1}A = I$ 를 둘 다 만족하는 행렬은 정방에만 존재**.

### 2-O-2. 좌역행렬 (Left Inverse)

$BA = I$ 를 만족하는 $B$.
- **존재 조건**: $A$가 $m \times n$, $m > n$ (세로로 긴), **full column rank ($r = n$)**
- **공식**: $B = (A^T A)^{-1} A^T$ — $A^T A$가 $n \times n$ 정칙이라 역행렬 존재
- $B$의 크기: $n \times m$
- ⚠ **유일하지 않음** — 여러 좌역행렬이 가능 (정방 $A^{-1}$과의 결정적 차이)

### 2-O-3. 우역행렬 (Right Inverse)

$AC = I$ 를 만족하는 $C$.
- **존재 조건**: $A$가 $m \times n$, $m < n$ (가로로 긴), **full row rank ($r = m$)**
- **공식**: $C = A^T (A A^T)^{-1}$
- $C$의 크기: $n \times m$
- ⚠ **유일하지 않음**

### 2-O-4. 의사역행렬 (Moore-Penrose Pseudoinverse) $A^+$

**어떤 행렬에도 존재**하는 가장 일반화된 역 — 비정방이든 rank-deficient든 항상 정의됨.

| 상황 | $A^+$ 공식 | 기존 개념과의 관계 |
|:---|:---|:---|
| 정방 + 정칙 | $A^{-1}$ | 일반 역행렬과 일치 |
| full column rank ($m > n$) | $(A^T A)^{-1} A^T$ | 좌역행렬과 일치 |
| full row rank ($m < n$) | $A^T (A A^T)^{-1}$ | 우역행렬과 일치 |
| 일반 (rank-deficient 포함) | SVD로 $A = U\Sigma V^T$일 때 $A^+ = V\Sigma^+ U^T$ | — |

### 2-O-5. 기하학적 직관

- **정방·정칙**: $\mathbb{R}^n \to \mathbb{R}^n$, 일대일 → 완벽히 되돌림
- **가로 길게 ($m < n$)**: 고차원을 저차원으로 압축 → 어떤 원본 고를지 모호 → 우역 여러 개
- **세로 길게 ($m > n$)**: 저차원을 고차원에 매립 → 도착 공간 일부만 상(image) → 좌역 여러 개

### 2-O-6. ML/AI 응용 연결

- **최소제곱법 (least squares)**: $A\mathbf{x} = \mathbf{b}$가 해 없을 때 $\|A\mathbf{x}-\mathbf{b}\|$ 최소화 → $\mathbf{x}^* = A^+ \mathbf{b}$
- **선형회귀 정규방정식**: $\hat{\beta} = (X^T X)^{-1} X^T y = X^+ y$ — 정확히 좌역행렬 공식
- 즉 ML에서 수없이 등장하는 $(X^T X)^{-1} X^T$는 "$X$의 좌역행렬" 이름만 다른 것.

### 2-O-7. 시험 범위 핵심

> **학부 선형대수에서 "역행렬"이라 하면 거의 항상 정방 기준.**
> 비정방이 문제에 나오면 (a) 문제 오류거나 (b) pseudoinverse 요구.

## 2-M. Ch2 자주 헷갈리는 것

| ❌ 헷갈림 | ✅ 바로잡기 |
|:---|:---|
| $(AB)^{-1} = A^{-1}B^{-1}$ | **틀림**. $(AB)^{-1} = B^{-1}A^{-1}$ — **역순** |
| $(AB)^T = A^T B^T$ | **틀림**. $(AB)^T = B^T A^T$ — **역순** |
| 피벗이 0이면 $A$는 항상 특이 | 행 교환하면 가역일 수 있음 ($PA = LU$) |
| $A = LU$ 하면 $L$ 계산이 복잡 | **승수 $l_{ij}$가 그 자리에 그대로** — 별도 계산 불필요 |
| $L$은 대각이 피벗 | $L$의 대각은 **모두 1**. 피벗은 $U$의 대각 |
| $A^{-1}$ 구해서 $\mathbf{x}=A^{-1}\mathbf{b}$가 효율적 | **낭비**. 소거로 바로 풀 것 |
| "$R_i + 2R_j$ 했으니 승수는 $+2$" | 승수는 **빼기 기준**. $R_i + 2R_j$면 $l_{ij} = -2$ |
| "$3\times 3$에서 두 열이 서로 배수 아니면 독립" | **아님**. 세 열 조합으로 종속일 수 있음 |
| $A^T$ (대각선 뒤집기) $= A^{-1}$ | 일반적으로 **다름**. 직교행렬일 때만 $A^T = A^{-1}$ |
| $A^{-1}$의 원소가 $A$ 원소와 단순 대응 | **무관**. 전체 계산 필요 |
| Singular면 $LU$ 아예 불가 | 소거가 진행되면 $LU$ 형태는 나옴. 단 $U$에 0 대각 생겨 실용성 없음 |
| "오차에 $h$ 있으면 1차 정확도" | $h$의 **최저 차수**로 판단 |
| 가우스-조르단은 "(1,1)만 1로 만들고 시작" | **대각선 따라 $n$번** 피봇 만들기. 매 스텝마다 피봇 확보 → 1로 → 열 소거 |
| 전치는 정방행렬에만 됨 | **어떤 행렬이든 가능**. 역행렬과 달리 조건 없음 |
| 비정방 행렬도 일반 역행렬 있을 수 있음 | **없음**. 좌역/우역/pseudoinverse만 가능 |
| 좌역행렬 / 우역행렬은 유일함 | **여러 개 존재 가능**. 정방 $A^{-1}$이 유일한 것과 대조 |
| 영행렬이 아니면 전치해도 변하지 않음 | 전치하면 **크기가 $n\times m$으로 바뀜** (비정방일 때). 성분도 재배열 |
